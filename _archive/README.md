Industrial Ink Website - Quick Start Guide

What's included:
- index.html (Home)
- products.html (Products list)
- product-wb-220.html (Product detail example)
- about.html (About)
- contact.html (Contact form)
- styles.css (site styles)
- assets/ (placeholder images)

Step 1 — Get a domain (optional but recommended)
- Buy a domain from Namecheap, GoDaddy, or your preferred registrar.

Step 2 — Create a GitHub account
- Go to https://github.com and sign up.

Step 3 — Create a new repository and upload files
- Create a new repository (public is fine).
- Upload all files from this package (or drag-and-drop on GitHub).

Step 4 — Deploy to Vercel (free)
- Sign up at https://vercel.com (use GitHub login).
- Click "New Project" → Import Git Repository → select your repo.
- Vercel will auto-detect a static site and deploy.

Step 5 — Bind your domain (optional)
- In Vercel dashboard, go to Settings → Domains → Add.
- Follow Vercel DNS instructions (add A/CNAME records in your registrar dashboard).
